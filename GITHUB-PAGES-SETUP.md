# GitHub Pages Setup

Your repository has been successfully pushed to GitHub!

## Enable GitHub Pages

1. Go to your repository: https://github.com/mahoney-jones/investment-calc
2. Click the **"Settings"** tab
3. Scroll down to **"Pages"** in the left sidebar
4. Under **"Source"**, select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **"Save"**

## Your Live Site

After GitHub Pages is enabled (may take a few minutes), your site will be available at:

**https://mahoney-jones.github.io/investment-calc/**

## Repository URL

**Repository**: https://github.com/mahoney-jones/investment-calc
